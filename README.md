# SQL Employee Management Database

## Project Overview
This project demonstrates the use of SQL to manage and analyze employee data. It includes creating tables, inserting records, and writing queries to retrieve meaningful business insights. 

## Objective
* Practice SQL database creation and management.
* Learn to retrieve and analyze data using SQL queries.
* Improve problem-solving skills through data analysis.

## Tools Used
* SQL

## Dataset
* Employee ID
* Employee Name
* Department
* Monthly Salary
* Joining Date
* Employment Status

## SQL Concepts Used
* CREATE TABLE
* INSERT INTO
* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
* UPDATE
* DELETE
  
## Sample Business Questions Answered
* How many employees are in each department?
* Which employee has the highest salary?
* What is the average salary by department?
* Which employees joined after a specific date?
* How many active employees are there?

## Files Included
* `Employee_Database.sql` – SQL script containing table creation, data insertion, and queries.
* `Employee_Dataset.xlsx` – Sample dataset used for the project.

## Skills Demonstrated
* SQL Database Management
* Data Retrieval
* Data Filtering
* Aggregate Functions
* Query Optimization
* Business Data Analysis
